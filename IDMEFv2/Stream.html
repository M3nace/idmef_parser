Attributes

offsetunit (Optional)

Rank Keyword Description
0 line Offset is a count of lines.
1 binary Offset is a count of bytes.

offset (Optional)

Amount of units (determined by the offsetunit attribute) to seek into the File

data (Required)

Content of the file
