Aggregates

SameDomainContact (0..1)

A domain name already cited in this document or through previous exchange that contains the identical contact information as the domain name in question. The domain contact information associated with this domain should be used instead of an explicit definition with the Contact class.

Contact (1..*)

Contact information for the domain. See Section 3.9.
