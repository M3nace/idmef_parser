Aggregates

Certificate (1..*)

A description of an X.509 certificate or certificate chain. See Section 3.24.1.

restriction (0..1)

See Section 3.3.1.

ext-restriction (0..1)

A means by which to extend the restriction attribute. See Section 5.1.1.

observable-id (0..1)

See Section 3.3.2.
