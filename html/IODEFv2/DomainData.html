Aggregates

Name (1..1)

The domain name of a system.

DateDomainWasChecked (0..1)

A timestamp of when the domain listed in the Name class was resolved.

RegistrationDate (0..1)

A timestamp of when domain listed in the Name class was registered.

ExpirationDate (0..1)

A timestamp of when the domain listed in the Name class is set to expire.

RelatedDNS (0..*)

Additional DNS records associated with this domain.

Nameservers (0..*)

The nameservers identified for the domain listed in the Name class. See Section 3.19.1.

DomainContacts (0..1)

Contact information for the domain listed in the Name class supplied by the registrar or through a whois query.

system-status (0..1)

A means by which to extend the system-status attribute. See Section 5.1.1.

domain-status (0..1)

A means by which to extend the domain-status attribute. See Section 5.1.1.

observable-id (0..1)

See Section 3.3.2.
