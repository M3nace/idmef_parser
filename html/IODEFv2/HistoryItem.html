HistoryItem

The HistoryItem class is an entry in the History (Section 3.13) log that documents a particular action or event that occurred in the course of handling the incident. The details of the entry are a free-form text description, but each can be categorized with the type attribute.

Aggregates

DateTime (1..1)

A timestamp of this entry in the history log.

IncidentID (0..1)

In a history log created by multiple parties, the IncidentID provides a mechanism to specify which CSIRT created a particular entry and references this organization's tracking number. When a single organization is maintaining the log, this class can be ignored. See Section 3.4.

Contact (0..1)

Provides contact information for the entity that performed the action documented in this class. See Section 3.9.

Description (0..*)

A free-form text description of the action or event.

DefinedCOA (0..*)

An identifier meaningful to the sender and recipient of this document that references a course of action (COA). This class MUST be present if the action attribute is set to "defined-coa".

AdditionalData (0..*)

A mechanism by which to extend the data model.

action (0..1)

A means by which to extend the action attribute. See Section 5.1.1.

restriction (0..1)

See Section 3.3.1.

ext-restriction (0..1)

A means by which to extend the restriction attribute. See Section 5.1.1.

observable-id (0..1)

See Section 3.3.2.
