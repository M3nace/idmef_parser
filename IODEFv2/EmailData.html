Aggregates

EmailTo (0..*)

The value of the "To:" header field (Section 3.6.3 of [RFC5322]) in an email.

EmailFrom (0..1)

The value of the "From:" header field (Section 3.6.2 of [RFC5322]) in an email.

EmailSubject (0..1)

The value of the "Subject:" header field in an email. See Section 3.6.5 of [RFC5322].

EmailX-Mailer (0..1)

The value of the "X-Mailer:" header field in an email.

EmailHeaderField (0..*)

The header name and value of an arbitrary header field of the email message. The name attribute MUST be set to the header name. The header value MUST be set in the element body. The dtype attribute MUST be set to "string".

EmailHeaders (0..1)

The headers of an email message.

EmailBody (0..1)

The body of an email message.

EmailMessage (0..1)

The headers and body of an email message.

HashData (0..*)

Hash(es) associated with this email message. See Section 3.26.

SignatureData (0..*)

Signature(s) associated with this email message. See Section 3.27.

observable-id (0..1)

See Section 3.3.2.
