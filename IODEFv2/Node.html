Aggregates

DomainData (0..*)

The domain (DNS) information associated with this node. If an Address is not provided, at least one DomainData MUST be specified. See Section 3.19.

Address (0..*)

The hardware, network, or application address of the node. If a DomainData is not provided, at least one Address MUST be specified. See Section 3.18.1.

PostalAddress (0..1)

The postal address of the node.

Location (0..*)

A free-form text description of the physical location of the node. This description may provide a more detailed description of where at the address specified by the PostalAddress class this node is found (e.g., room number, rack number, or slot number in a chassis).

Counter (0..*)

A counter with which to summarize properties of this host or network. See Section 3.18.3.
